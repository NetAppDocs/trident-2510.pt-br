== A restauração de backups Restic de arquivos grandes pode falhar

Ao restaurar arquivos de 30GB ou maiores de um backup do Amazon S3 feito usando Restic, a operação de restauração pode falhar. Como solução alternativa, faça backup dos dados usando Kopia como o data mover (Kopia é o data mover padrão para backups). Consulte  https://docs.netapp.com/us-en/trident/trident-protect/trident-protect-protect-apps.html["Proteja aplicativos usando Trident Protect"^] para instruções.
